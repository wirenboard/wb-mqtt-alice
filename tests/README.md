
## microsoft test run

```powershell
> python3 -m pip install -U pytest
> python3 -m pytest -q
```

Тут есть юнит тесты отдельных частей системы

1) Сначала отдельно тестируется работа классов которые формируют структуры
   данных при чтении конфига, это два класса:
   - Регистри для девайсов
   - Регистри для биндинга
2) Далее тестируется чтение конфига
   - Сначала формирование через передачу JSON сформированного в ОЗУ
   - Далее реальное чтение фала
3) Потом как отдельная система самодостаточно тестируются downstream адаптеры
   Тут можно проверить их работу, как они генерируют сообщения и тд
4) Далее тестируется роутер
   Роутер совмещает в себе две функции
   - прием сообщений от адаптеров
   - нормализацию данных во внутренний формат ядра интеграции
5) Потом тестируется уже в сборе роутер и адаптеры с конфигом в сборе -
