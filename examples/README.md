# Папка с примерами использования

Это **не production-код** !!

В данной папке собраны примеры использования частей системы.

## Назначение

Это не замена unit тестам, а дополнение к ним. Примеры можно использовать\
для интеграктивного исследования и тестов системы:

- понятно как их можно использовать в минимальном варианте каждую часть
- поэксперементировать с каждым куском системы по отдельности используя
уже написанный базовый пример, а не разбираясь по исходникам

## Минимальный состав примера

Каждый пример должен быть в отдельной папке, внутри которой минимально
содержится два файла:

- run.py - файл примера который можно запустить в терминале и посмотреть на его работу
- README.md - файл с описанием что это за пример, как его запустить и использовать

Каждый пример должен запускаться из корня репозитория командой вида:

```powershell
> python3 -m examples.<!example_folder!>.run
```

Однотипная структура примеров сильно упрощает их использование новыми людьми.
